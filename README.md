# Prog2WebApi

An educational project used to teach 12th graders web API development.

## Live Demo

The API is deployed and accessible at:
- API: https://prog2-webapi-teaching-demo.onrender.com
- Swagger: https://prog2-webapi-teaching-demo.onrender.com/swagger

## Tech Stack

- ASP.NET Core 9.0
- Entity Framework Core
- SQLite
- JWT Authentication
- Swagger/OpenAPI
- Docker (for deployment)

## Running Locally

```bash
cd Prog2WebApi
dotnet run
```

- API: `http://localhost:5153`
- Swagger: `http://localhost:5153/swagger`

## Deployment

The application is deployed to [Render](https://render.com) using Docker.
