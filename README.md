# Prog2WebApi

An educational project used to teach 12th graders web API development.

## Tech Stack

- ASP.NET Core 9.0
- Entity Framework Core
- SQLite
- JWT Authentication
- Swagger/OpenAPI

## Run

```bash
cd Prog2WebApi
dotnet run
```

API: `http://localhost:5153`
Swagger: `http://localhost:5153/swagger`
